Upper Garden Railing and Fence
====================================================================
The Upper Garden is the area above the wooden retaining wall at the back of the studio.
Currently there is no fence which makes it potentially dangerous for anyone walking or working up there.
Type of fence still to be determined, but it should not be visible when looking up from inside the studio.
